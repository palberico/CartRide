# CartRide — Planned Features

## High Priority
- [ ] Push notifications — notify riders when a driver accepts; notify drivers of new ride requests
- [ ] Two-way ratings — riders rate drivers, drivers rate riders after each completed ride
- [ ] In-app navigation for drivers — "Navigate to pickup" button that opens Apple/Google Maps

## Medium Priority
- [ ] Ride history — riders see past rides; drivers see completed trips and an informal earnings total
- [ ] Favorite/saved locations — one-tap pickup and destination presets
- [ ] Scheduled rides — book a ride in advance for a specific time or event
- [ ] Admin ride dashboard — live view of all active/pending rides, stats, and driver activity
- [ ] Add more cities — expand service beyond Daybreak to other communities

## Longer Term
- [ ] PWA / installable — web app manifest so riders can add CartRide to their home screen
- [ ] SMS fallback via Twilio — text drivers when a new ride request comes in
